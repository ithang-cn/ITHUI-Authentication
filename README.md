## 介绍

* ITHUI-Authentication 适用于上传身份认证、实名认证、完善认证资料，兼容小程序、APP、H5，三款样式按需选择。
* ITHUI-Authentication 目前仅完成样式一，其他样式开发中，敬请期待。

## 版本

v1.0.0

## 特性

* ITHUI采用`uniapp`制作，基于`uView 2.x`组件，兼容小程序、APP、H5；
* 目前仅测试了APP、微信小程序、QQ小程序、H5。
